# trihardgaming.net

TriHard Gaming Tech — AI & Tech Insights website hosted on WordPress via Hostinger.

## Architecture

```
trihardgaming/
├── index.php            # WordPress theme entry point
├── style.css            # Theme stylesheet + metadata
├── functions.php        # Theme setup, menus, customizer
├── screenshot.png       # Theme preview image
├── parts/
│   ├── header.php       # Site header
│   ├── footer.php       # Site footer
│   └── sidebar.php      # Sidebar widget area
├── templates/
│   ├── front-page.php   # Homepage layout
│   ├── single.php       # Single post
│   └── archive.php      # Archive/listing
└── assets/
    ├── css/
    │   └── style.css    # Compiled styles
    ├── js/
    │   └── main.js      # Interactive scripts
    └── images/
```

## Quick Start

1. Upload via Hostinger File Manager to `/public_html/wp-content/themes/trihardgaming/`
2. Activate in **Appearance → Themes**
3. Set **Settings → Reading** → Static front page

## Content Pipeline

Daily articles are auto-generated by the Hermes content pipeline and pushed to this repo.

- **Blog posts**: AI productivity, tech insights, remote work trends
- **Social content**: Snippets for distribution
- **Newsletter**: Weekly roundups

## Deployment

```bash
# Push to GitHub
git add -A && git commit -m "update" && git push origin main
```

WordPress pulls from this repo via Hostinger hosting.

`trihardgaming.net` → Hostinger → WordPress
